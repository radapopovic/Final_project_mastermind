Rules

South Park Mastermind is version of a Mastermind, code-breaking game based on an older, paper game called Bulls and Cows. Try to guess the pattern, in both order and character, within six turns. After submitting a row, a green check mark is placed for each code peg from the guess which is correct in both character and position. A white check mark indicates the existence of a correct character code peg placed in the wrong position. If you want to learn basic principles of scientific reasoning and develop your logic skills, raise this game to a higher level, follow this link.

This is an implementaion of a game Mastermind that I have made as a student for my final project of Front-end programming course at Comtrade academy. All images and web-materials are used for educational purposes only.
